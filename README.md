# Michi XO – Tic Tac Toe for Wear OS

A kawaii-inspired Tic Tac Toe game designed for smartwatches.

## Features
- Single player mode
- Clean minimalist UI
- Optimized for small screens
- Lightweight and smooth

## Tech Stack
- Kotlin
- Wear OS
- Android Studio

## Installation
Clone the repository and open in Android Studio.

## Future Improvements
- Difficulty levels
- Sound effects
- Score tracking
